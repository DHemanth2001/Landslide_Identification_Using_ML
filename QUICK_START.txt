## 🔍 Implementation Details

### Phase 1: CNN Ensemble
- Models: ConvNeXt-CBAM-FPN, SwinV2, EfficientNetV2
- Input: Satellite image (224×224)
- Output: Landslide/Not + confidence

### Phase 2: Regional HMM Forecast
- Data: NASA Global Landslide Catalog (11,033 events)
- Search: ±100km radius around location
- Output: Type, probability, peak month, 3-step forecast

**Conditional:** Phase 2 ONLY runs if Phase 1 detects landslide
